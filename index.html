DESAFIO PROPOSTO NO BOOTCAMP DA DIO
OBJETIVOS:

• Criar um contador de incrementos utilizando o método AddEventListener nos botões;

• Implementar regras de mudança de atributos manipulando o DOM (Document Object Model) com os métodos document.GetElement;

APRIMORAMENTOS:

• Dial analógico de 03 dígitos;
• Botão de troca de temas;
• Botão de Reset do contador;
• Chave seletora de modos:
  Mode 01: O display muda para vermelho e o dial desabilita quando os números são negativos;
  Mode 02: O contador desabilita os botões de - ou + quando o limite é atingido.
